Week 1：LLM 基础 + Prompt 工程（14小时）
  - 把 System Prompt 设计搞明白
  - 理解 messages 数组的多轮对话机制
  - 实验：用不同 Prompt 让同一问题得到不同结果
  交付物：一个"有人格"的对话助手

Week 2：Tool Calling 实战（14小时）
  - 理解 Function Calling 协议
  - 实现至少 2 个真实工具（搜索/天气/计算）
  - 让模型自主选择调用哪个工具
  交付物：一个能查天气、能做计算的小 Agent

Week 3：RAG 入门（14小时）
  - Embedding 概念 + 向量相似度
  - 用 Chroma 或 LanceDB 做简单文档检索
  - 把检索结果注入 Prompt
  交付物：一个能回答"基于某文档"问题的问答系统

Week 4：整合 + 内容创作场景（14小时）
  - 把前三周能力合并成一个内容创作 Agent
  - 对标 pagepop.ai 的场景做针对性设计
  - 写清楚你的设计文档（面试时讲）
  交付物：可演示的内容创作 Agent Demo
